**DSS — Directory Size Scanner** é uma ferramenta que analisa um diretório e exibe todas as pastas em ordem decrescente de tamanho, calculado em megabytes (MB).

Desenvolvi esta aplicação para solucionar uma limitação que encontrei no Windows 10. Ao tentar liberar espaço em disco, era difícil identificar rapidamente quais jogos ou programas ocupavam mais armazenamento, pois o Explorador de Arquivos do Windows permite ordenar arquivos por tamanho, mas não pastas.

Com o DSS, é possível visualizar facilmente quais diretórios consomem mais espaço, facilitando a decisão sobre quais arquivos, jogos ou aplicações remover para recuperar armazenamento.
