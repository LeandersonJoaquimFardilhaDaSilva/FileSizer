# português
**DSS — Directory Size Scanner** é um utilitário para Windows que escaneia diretórios e lista as pastas em ordem decrescente de tamanho. O objetivo é fornecer uma visão rápida e organizada do consumo de armazenamento em disco.

O projeto surgiu da necessidade de identificar quais jogos, programas e diretórios ocupavam mais espaço no sistema. Embora o Windows permita ordenar arquivos por tamanho, ele não oferece a mesma funcionalidade para pastas de forma prática. O DSS preenche essa lacuna, tornando o gerenciamento de armazenamento mais simples e eficiente.

# english
**DSS — Directory Size Scanner** is a Windows utility that scans directories and lists folders in descending order by size. Its purpose is to provide a quick and organized overview of disk space usage.

The project was created to address a common storage management challenge: identifying which games, programs, and directories occupy the most space on a system. While Windows allows files to be sorted by size, it does not offer a practical way to do the same for folders. DSS fills this gap by making disk space analysis and storage management simpler and more efficient.
